#  Sistema de puntos
El siguiente es el código de un sistema de puntos orientado a consultorio de medicina estética, pero puede adaptarse otros usos.

Las principales funcionalidades planteadas son las siguientes:

1. Dar de altas de pacientes o clientes
2. Sumar puntos con las compras
3. Canjear puntos acumulados
4. Definir premios a canjear
5. Posibidad de efectuar vencimientos de puntos.


El sistema esta realizado con Drupal versión 7.
